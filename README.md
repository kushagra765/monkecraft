# Monkecraft
A Minecraft (R)-like game.
## Building:
### Dependencies:
1. [`raylib`](https://github.com/raysan5/raylib)
2. `make`
3. `gcc`
### Building (For real now)
```sh
git clone https://github.com/monkelab/monkecraft/
cd monkecraft
make all
```
## Running
There ain't much rn but here:
```sh
# In monkecraft directory
./monkecraft
```
